# Instructions pour ajouter l'image d'avatar

Pour que l'image d'avatar s'affiche dans le jeu, placez votre image dans le dossier du jeu avec l'un de ces noms :

- `avatar.png`
- `avatar.jpg`
- `avatar.jpeg`
- `cat_ghost.png`
- `cat_ghost.jpg`

L'image sera automatiquement chargée et affichée dans le menu Avatar.

## Format recommandé

- Format : PNG ou JPG
- Taille : L'image sera automatiquement redimensionnée pour s'adapter à l'écran
- Nom : `avatar.png` (recommandé)
